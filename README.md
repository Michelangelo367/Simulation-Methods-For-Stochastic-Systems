# Simulation-Methods-For-Stochastic-Systems

## Requirements
* Python 2.7 
* Numpy >= 1.14.2
* Matplotlib >= 2.2.0
* Pandas >= 0.22.0
* Scikit-Learn >= 0.19.1
* Networkx >= 2.1

## Usage

```
$ cd /path/to/folder 
$ python main.py 
```
